Default Page

IT WORKS!

This is the default welcome page used to test the correct operation of the Tailwind after installation on systems. It is based on the equivalent page on Debian, from which the Ubuntu Apache packaging Is derived. If you can read this page, it means that the Tailwind services installed at this site is working properly. You should replace this file (located at ./public/index.html) before continuing to operate your Tailwind.

If you are a normal user of this web site and don't know what this page is about, this probably means that the site is currently unavailable due to maintenance. If the problem persists, please contact the site's administrator.

CONFIGURATION OVERVIEW

Tailwind default configuration is different from the upstream default configuration, and split into several files optimized for interaction with Tailwind tools. The configuration system is fully documented in this page Refer to this for the full documentation. The configuration layout for an Tailwind installation on systems is as follows: 

                ____________________________________________________________________________________
                | -> ./public/index.html
                | -> ./public/css/tailwind.css
                | -> ./tailwind.css
                | $ npm init -y
                | $ npm install -D tailwindcss
                | $ npx tailwindcss init
                |
                 \
                  | conf tailwind.config.js > content
                  |
                 /
                |
                | In -> ./tailwind.css
                ------------------------------|
                |   @tailwind base;           |
                |   @tailwind components;     |
                |   @tailwind                 |
                ------------------------------|
                |
                 \
                  |
                  |
                 /
                |   href="css/tailwind.css" rel="stylesheet"
                |
                |
                | $ npx tailwindcss -i ./tailwind.css -o ./public/css/tailwind.css --watch
                |____________________________________________________________________________________
